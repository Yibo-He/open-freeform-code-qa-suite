Sure, you can use the `WindowMetrics` class to get the window's visible size, including insets such as the navigation bar. Here's how you can modify your function:

```kotlin
fun getWidthHeight(wm: WindowManager): Int {
    val windowMetrics = wm.currentWindowMetrics
    val windowInsets: WindowInsets = windowMetrics.windowInsets

    val insets = windowInsets.getInsetsIgnoringVisibility(WindowInsets.Type.navigationBars() or WindowInsets.Type.displayCutout())
    val insetsHeight = insets.top + insets.bottom

    val b = windowMetrics.bounds
    val orig_height = b.height()

    // Subtract the insets from the original height to get the actual screen height
    val actual_height = orig_height - insetsHeight

    return actual_height
}
```

This function will return the actual height of the screen, excluding the navigation bar and display cutout (if any).
