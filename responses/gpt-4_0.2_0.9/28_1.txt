```python
import plotly.figure_factory as ff
import kaleido

def plot_confusion_matrix(matrix: list[list[int]], x_labels: list[str], y_labels: list[str]):
    # Create the heatmap
    fig = ff.create_annotated_heatmap(matrix, x=x_labels, y=y_labels, colorscale='Viridis')
    
    # Update layout
    fig.update_layout(
        title='Confusion Matrix',
        xaxis=dict(title='Predicted Label'),
        yaxis=dict(title='True Label'),
    )
    
    # Add annotations
    for i, row in enumerate(matrix):
        for j, value in enumerate(row):
            fig.add_annotation(
                x=j,
                y=i,
                text=str(value),
                showarrow=False,
                font=dict(
                    color="black",
                    size=12
                )
            )
    
    # Update yaxis to reverse order
    fig.update_yaxes(autorange="reversed")
    
    return fig
```

This function creates an annotated heatmap using the provided matrix and labels. The colorscale 'Viridis' is used for the heatmap. The layout of the figure is updated to include a title and labels for the x and y axes. Annotations are added for each cell in the matrix, displaying the corresponding value. The y-axis is reversed to match the typical layout of a confusion matrix. The figure is then returned.
