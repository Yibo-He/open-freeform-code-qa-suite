```python
import numpy as np
from matplotlib import pyplot as plt

def draw_histogram():
  x = np.random.randn(100000)
  
  # Create a histogram with 20 bins
  plt.hist(x, bins=20, edgecolor='black')

  # Set the title and labels
  plt.title('Histogram of Normal Distribution')
  plt.xlabel('Value')
  plt.ylabel('Frequency')

  # Save the figure
  plt.savefig('figure.pdf')

# Call the function
draw_histogram()
```

In this code, `np.random.randn(100000)` generates 100,000 random numbers from a standard normal distribution. `plt.hist(x, bins=20, edgecolor='black')` creates a histogram of these numbers with 20 bins. The edgecolor parameter is used to make the bins more visible. `plt.savefig('figure.pdf')` saves the figure as a PDF file.
