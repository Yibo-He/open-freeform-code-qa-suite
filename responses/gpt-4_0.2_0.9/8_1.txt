Sure, here is a simple function that converts a local Date object to a UTC Date object in JavaScript:

```javascript
function convertToUTC(date) {
    // Get the local time zone offset in minutes
    let timezoneOffset = date.getTimezoneOffset() * 60 * 1000;

    // Subtract the time zone offset from the local date and create a new Date object
    let utcDate = new Date(date.getTime() - timezoneOffset);

    return utcDate;
}
```

You can use this function by passing a Date object to it. The function will return a new Date object that represents the same moment in time, but in UTC rather than the local time zone.
