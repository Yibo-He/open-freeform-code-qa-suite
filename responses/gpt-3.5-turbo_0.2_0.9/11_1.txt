Sure, here's an implementation of the TRIM function in SQL that you can use:

```
CREATE FUNCTION TRIM(@string VARCHAR(8000))
RETURNS VARCHAR(8000)
BEGIN
    WHILE CHARINDEX(' ', @string) = 1
        SET @string = SUBSTRING(@string, 2, LEN(@string) - 1)

    WHILE CHARINDEX(' ', @string, LEN(@string) - 1) > 0
        SET @string = LEFT(@string, LEN(@string) - 1)

    RETURN @string
END
```

You can then use the TRIM function in your code like this:

```
DECLARE @ss varchar(60)
SET @ss = ' admin '

SELECT TRIM(@ss)
```

This will return 'admin' with no leading or trailing spaces.
