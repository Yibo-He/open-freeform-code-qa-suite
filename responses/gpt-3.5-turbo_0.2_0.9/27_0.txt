Make sure that you have imported `FlexLayoutModule` in your `login-view` module as well. Here is an example:

```
import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { LoginComponent } from './login/login.component';
import { MaterialModule } from 'src/app/material/material.module';
import { FlexLayoutModule } from '@angular/flex-layout';

@NgModule({
  declarations: [LoginComponent],
  imports: [
    CommonModule,
    MaterialModule,
    FlexLayoutModule
  ]
})
export class AuthModule { }
```

Also, make sure that you have added the `fxLayout` directive to the HTML element in your `login-view` component. Here is an example:

```
<div fxLayout="row" fxLayoutAlign="center center">
  <mat-card>
    <mat-card-header>
      <mat-card-title>Login</mat-card-title>
    </mat-card-header>
    <mat-card-content>
      <!-- Login form goes here -->
    </mat-card-content>
  </mat-card>
</div>
```

This will create a row layout with center alignment for the `mat-card` element.
