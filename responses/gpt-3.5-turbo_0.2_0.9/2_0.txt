Yes, you can use the Matplotlib inline backend to display the plots directly in the Visual Studio Code editor. Here's how you can modify your code to achieve this:

```
import matplotlib.pyplot as plt
import matplotlib as mpl
import numpy as np

%matplotlib inline  # Add this line to enable inline plotting

x = np.linspace(0, 20, 100)
plt.plot(x, np.sin(x))
plt.show()
```

Note that the `%matplotlib inline` line should be added at the beginning of the code. This line enables the inline backend, which will display the plots directly in the editor.
