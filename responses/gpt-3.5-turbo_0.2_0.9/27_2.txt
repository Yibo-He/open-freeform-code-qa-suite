Make sure that you have imported `FlexLayoutModule` in the `login-view` module as well. Here is an example of how to import it:

```
import { NgModule } from '@angular/core';
import { CommonModule } from '@angular/common';
import { LoginComponent } from './login/login.component';
import { MaterialModule } from '../../material/material.module';
import { FlexLayoutModule } from '@angular/flex-layout';

@NgModule({
  declarations: [LoginComponent],
  imports: [
    CommonModule,
    MaterialModule,
    FlexLayoutModule
  ]
})
export class AuthModule { }
```

Also, make sure that you have added the `fxLayout` directive to the HTML template of the `login-view` component. Here is an example:

```
<div fxLayout="column" fxLayoutAlign="center center">
  <mat-card>
    <mat-card-header>
      <mat-card-title>Login</mat-card-title>
    </mat-card-header>
    <mat-card-content>
      <!-- Login form goes here -->
    </mat-card-content>
  </mat-card>
</div>
```

With these changes, `angular/flex-layout` should work in your `login-view` component.
