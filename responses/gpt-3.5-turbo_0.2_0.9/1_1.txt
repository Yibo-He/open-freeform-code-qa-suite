const today = new Date();
const month = today.getMonth() + 1;
const day = today.getDate();
const year = today.getFullYear();
return `${month}/${day}/${year}`;
} 

This function uses the `Date` object in Javascript to get the current date. It then extracts the month, day, and year from the `Date` object and formats them into a string of the desired format.
